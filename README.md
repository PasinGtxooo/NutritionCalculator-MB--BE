# Nutrition Calculator MB - Backend

Backend API สำหรับระบบ Nutrition Calculator พัฒนาด้วย Spring Boot  
ทำหน้าที่จัดการข้อมูลผู้ใช้ การบันทึกอาหาร การวิเคราะห์โภชนาการ และการเชื่อมต่อฐานข้อมูล

โปรเจคนี้เป็นส่วน **Backend** ของระบบ Nutrition Calculator ซึ่งทำงานร่วมกับ Frontend ที่พัฒนาแยก repository

---

## Features

- ระบบ Authentication (Register / Login)
- บันทึกข้อมูลอาหารของผู้ใช้ (Food Logging)
- วิเคราะห์โภชนาการจากข้อมูลอาหาร
- รองรับการอัปโหลดรูปภาพอาหาร
- ระบบ Chatbot สำหรับแนะนำอาหาร
- API สำหรับ Dashboard แสดงข้อมูลโภชนาการ
